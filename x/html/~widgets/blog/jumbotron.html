What's New in SuperDom!  version X

Specially for Light Blue 2.1 we have developed a plugin that allows to easily implement basic widget functions that lots of our customers have requested. The plugin is called Widgster. For now it has the following essential widget features:

  • Collapse/Expand - all widgets can be collapsed to fill only header's vertical space;
  • Close - closable. Any widget may be removed by clicking the close btn;
  • Full Screen - an option to make widget fill the whole window (just like OS);
  • Ajax Load - the hottest option allowing to load/reload widget content asynchronously. You just need to provide an url to fetch the data from. With loader delivered.

In the future this plugin may be released under MIT license, but for now it's available exclusively with Light Blue.
